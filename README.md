# Designed a Dashboard to Analyze Sales vs Profit

## Objective
To design a dashboard in Excel to show sales vs profit for product categories in terms of regions and months

## Process
• Checked the data to find missing/ null values
• Created a histogram for bin frequency 
• Created a table for month-wise sales vs profit
  • Used OFFSET then SUMIFS functions to copy, sort and rank the required values
• Created another table for region-wise sales vs profit
  • Used SUMIFS functions to copy, sort and rank the required values
• Created a user control combo box to show each category's sales vs profit
• Built charts to depict sales vs profit for each category/ visual representation of sales vs profit

## Conclusion/ Insights and Trends from Sales Data
• Sales and profit are high in each month for all product categories
• Highest sales and profits are earned by central region followed by south region for each product category
• Bar charts makes it easy to see the trends and patterns for each product category both region and month-wise 
• This concludes that central region in the world has the highest sales and generates the most profit for all product categories
